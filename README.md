# 🏦 Bank Database SQL Analysis

## 📌 Project Overview

This project demonstrates SQL skills through the analysis of a banking database using SQL Server. It includes database schema creation, data querying, filtering, joins, aggregations, and business-oriented SQL analysis to extract meaningful insights from banking data.

---

## 🛠 Tools Used

- SQL
- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
- Relational Database

---

## 📊 Database Structure

The project includes a complete banking database schema along with SQL queries developed to analyze customer, account, and transaction data.

---

## 📈 SQL Operations Performed

- Database creation
- Table creation
- Primary & Foreign Keys
- SELECT statements
- WHERE filtering
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Subqueries
- Data Retrieval
- Business Reporting Queries

---

## 🔍 Project Objectives

- Build a relational banking database.
- Practice SQL query writing.
- Analyze banking data using SQL.
- Retrieve meaningful business insights.
- Strengthen database design and analytical skills.

## ▶️ How to Run

1. Open Microsoft SQL Server Management Studio (SSMS).
2. Execute **02.BankProjectStructure.sql** to create the database structure.
3. Open **Məmmədli Lalə-SQL layihə.BANK.sql**.
4. Run the SQL queries to analyze the banking database.
